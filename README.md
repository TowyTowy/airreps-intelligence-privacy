# AirReps Intelligence — Privacy Policy

**Last updated:** June 11, 2026

AirReps Intelligence is a Discord bot that provides AI-powered answers and community features for the AirReps community. This policy explains what data the bot processes and stores.

## Data we process

- **Message content.** The bot reads messages in servers it is in and in direct messages. Message content is used to: (1) answer questions when the bot is mentioned, replied to, or messaged directly; (2) flag messages that may need moderator attention; (3) build a community knowledge base and a weekly activity digest for server staff.
- **User and server identifiers.** Discord user IDs, usernames, channel IDs, and server IDs are processed alongside messages.
- **Images.** Images attached to messages directed at the bot may be processed to generate a response.

## Third-party processing

To generate AI responses, relevant message content (and attached images) is sent to a third-party AI inference provider (OpenRouter, which routes to model providers such as Anthropic). This content is used only to generate the response. We do not sell user data or share it with third parties for advertising.

## Data we store

- **Conversation history** with the bot, to maintain context across a conversation.
- **Usage counters** (per-user message counts) for rate limiting.
- **Community knowledge entries and digest summaries** derived from server activity.
- **User-submitted reports and feedback.**

Stored data lives on a private server operated by the bot owner and is not publicly accessible.

## Data retention and deletion

Stored data is kept only as long as needed for the features above. To request deletion of your data, contact the bot owner (see below) with your Discord user ID, and associated stored data will be removed.

## Children

The bot is intended for users who meet Discord's minimum age requirements.

## Changes

We may update this policy; the "Last updated" date will reflect the current version.

Questions or deletion requests: contact the bot owner via the AirReps Discord server, or open an issue at the bot's support channel. Email: towy@airreps.link
